- Equality Operators: `==`, `===`, `!=`, `!==`
- Bitwise Operators: `&`, `|`, `^`, `~`, `<<`, `>>`
- Logical Operators: `&&`, `||`, `!`
- Ternary Operator: `condition ? expr1 : expr2`
- Operator Precedence
